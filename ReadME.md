# Freekout2.0 Readme

Freekout2.0 is a web application that serves as an ecosystem for social media influencers and brands. It allows influencers to connect with brands and collaborate on various projects. The project is built using HTML, CSS, and JavaScript, and it incorporates the Particle.js library for particle effects.

## Introduction

Freekout2.0 provides an interactive and visually appealing platform for influencers and brands to discover each other, establish connections, and collaborate on projects. The project utilizes Particle.js library to add engaging particle animations to the website, creating a dynamic user experience.

## Features

- **Navigation Bar**: The top navigation bar provides links to different sections of the website, including Brands, Influencers, Pricing, Blogs, and Contact.

- **Hero Section**: The hero section displays a captivating tagline and sub-title to attract users' attention. It also offers buttons for influencers and brands to join the platform.

- **Particle.js Animation**: The website incorporates Particle.js library to create visually appealing particle animations, enhancing the overall design.

- **Wave Animation**: The wave animation in the background adds a dynamic element to the website.

## Setup and Installation

To set up the Freekout2.0 project locally, follow these steps:

1. Clone the repository: [Link to Repository](https://github.com/yourusername/freekout2.0.git)

2. Navigate to the project directory.

3. Open the `index.html` file in your preferred web browser.

## Usage

Upon opening the `index.html` file in a web browser, you will be greeted with the Freekout2.0 landing page. Navigate through the website using the top navigation menu to explore different sections. Users can click on the "I'm an Influencer" or "I'm a Brand" buttons in the hero section to proceed with registration or learn more about the platform.

## Customization

If you wish to customize the Freekout2.0 project, you can modify the following files:

- `index.html`: Edit the content and structure of the website.

- `style.css`: Customize the styles, colors, and layout of the elements.

- `app.js`: Modify Particle.js settings or add additional interactivity to the website.

You can also replace the images used in the project with your own, but ensure that you have the appropriate permissions and licenses for using those images.
